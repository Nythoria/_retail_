Edit Macro Icons by ShaunDreclin
Twitter: @ShaunDreclin

Provides a basic slash command to edit macro, equipment set, and guild bank tab icons

Usage:
/emi <account/acc> <#> <icon name/id>
/emi <character/char> <#> <icon name/id>
/emi <equipmentset/equip> <#> <icon name/id>
/emi <guildbanktab/gbt> <#> <icon name/id>

Examples:
/emi account 1 ability_cheapshot
/emi equip 1 132089

Icon names and ids can be found at https://www.wowhead.com/icons
